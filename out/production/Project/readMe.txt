                        MORRIS GAME

To compile the source codes:
javac *.java

To run the program
java <Program> <InputFile> <depth> <OutputFile>
* args[0] - inputFile which is files/boardOpening.txt for opening phase of game
                               files/boardGame.txt for midgame and endgame phase
* args[1] - depth of the tree
* args[2] - outputFile which is files/boardMMO.txt for program MiniMaxOpening.java
                                files/boardMMOB.txt for program MiniMaxOpeningBlack.java
                                files/boardMMOI.txt for program MiniMaxOpeningImproved.java
                                files/boardMMG.txt for program MiniMaxGame.java
                                files/boardMMGB.txt for program MiniMaxGameBlack.java
                                files/boardMMGI.txt for program MiniMaxGameImproved.java
                                files/boardABO.txt for program ABOpening.java
                                files/boardABG.txt for program ABGame.java
